# 1. 커뮤니티 소개

본 커뮤니티는 웹으로 구현되었으며, 게시글을 볼 수 있고, 작성할 수 있습니다. 

# 2. 파일 소개

|  | 기능 소개 |
| --- | --- |
| auth/login | 로그인 페이지 |
| auth/password | 비밀번호 변경 페이지 |
| auth/profile | 닉네임 변경 페이지 |
| auth/register | 회원가입 페이지 |
| post/add | 게시물 작성 페이지 |
| post/detail | 게시물 상세 조회 페이지 |
| post/index | 게시물 목록 조회 페이지 |
| post/modify | 게시물 수정 페이지 |
